* AUR helper -- paru
Install AUR helper
Reuires =rust fakeroot pkgconfig=
#+begin_src sh
#install AUR helper\
doas sudo pacman -S --needed base-devel fakeroot rust pkgconfig
git clone https://aur.archlinux.org/paru.git
cd paru
makepkg -si
#+end_src
